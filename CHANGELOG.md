# 更新日志

## [1.0.0] - 2026-06-01

### ✨ 新增

- **Web 配网模式**：首次上电自动创建 `ESP32-LED-Clock` 热点，通过浏览器即可配置 WiFi，无需修改代码
- **WiFi 扫描功能**：配网页面支持扫描周围 WiFi 信号，点击即可选择网络
- **配置持久化存储**：WiFi 配置通过 `Preferences` 保存到 NVS 闪存，断电不丢失
- **BOOT 按键重新配网**：按住 BOOT 按键重新上电即可再次进入配网模式
- **4 种显示模式**：时钟模式、音乐频谱模式、俄罗斯方块时钟模式、变色时钟模式，每 30 秒自动轮播
- **NTP 自动校时**：连接 WiFi 后自动通过 NTP 服务器同步时间（默认使用阿里云 NTP）
- **DHT11 温湿度显示**：实时读取并显示温湿度数据
- **启动画面**：上电显示 ESP32 LED Clock 启动画面

### 📝 优化

- **README.md 重构**：优化文档结构，新增项目结构图、快速开始指南、WiFi 配置说明
- **图片本地化**：将驱动板实物图下载到本地 `images/` 目录，不再依赖外部链接
- **目录规范化**：资源目录从 `README.assets` 重命名为 `images`

### 🔧 技术细节

- 开发框架：Arduino IDE
- 核心库：`ESP32_HUB75_LED_MATRIX_PANEL_DMA_Display`
- 传感器：DHT11 温湿度传感器
- 时间同步：NTP (ntp.aliyun.com)
- 配网方式：WebServer + WiFi AP 模式
